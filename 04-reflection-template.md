**Tanggal Entry:** ~2025-XX-XX (avg beli Rp 106)
**Tanggal Exit (partial):** 2026-07-22
**Jenis:** Awalnya invest, jadi cutloss partial

### Entry (ORI)
- Harga: Rp 106,15 (avg)
- Total lot: 2,933
- Modal: Rp 31.136.434
- % portfolio: 42% (over-concentrated!)
- Alasan masuk: FOMO, tidak ada riset fundamental
- Setup: -
- Target profit: -
- Stop loss: - (tidak ditentukan!)
- Time horizon: unclear

### Kondisi Diri Saat Entry
- Emosi: FOMO
- Fisik: -
- Keuangan: -

### Exit Partial (50%)
- Harga: Rp 86
- Alasan keluar: Reduce over-concentration + insider selling CEO
- P/L: Realized -Rp 2.953.990 (-19%)
- Durasi hold: ~6-12 bulan

### Refleksi
- **Yang salah:**
  - Masuk tanpa riset fundamental
  - Position size 42% portfolio (2x lipat batas aman 15%)
  - Tidak set stop loss di awal
  - Beli SETELAH insider CEO jual di harga Rp 140
  - Ga cek: perusahaan tidak bagi dividen 2 tahun

- **Pelajaran:**
  - SELALU cek insider ownership sebelum beli
  - SELALU cek dividend history
  - SELALU set position size < 15%
  - SELALU set stop loss SEBELUM beli
  - Micro-small cap = risiko tinggi, harus lebih hati-hati

- **Skor eksekusi:**
  - Entry timing: 2/10 (FOMO, no research)
  - Exit timing: 6/10 (terlambat tapi berani cutloss)
  - Risk management: 1/10 (no stop loss, over-concentration)
  - Disiplin: 3/10 (belum punya rules)

### Catatan Emosi
- **Ketakutan:** Menyesal jual di harga rendah, takut IKAN naik lagi
- **Godaan:** Sempat mau average down ("kan sudah loss segini, tinggal turun sedikit")
- **Reaksi setelah exit:** Lega, meskipun perih. Cash bisa dipakai deploy blue chip.
